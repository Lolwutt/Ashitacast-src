Quick Start Directions

1.    Browse to your Ashita4 folder, enter the config folder, and create a new folder inside it named ashitacast.

2.    Copy your XML from Ashita3 to the new ashitacast folder in Ashita4.  Ensure the top level tag is <ashitacast>.

3.  Rename your Ashita3 XML to Charname_JOB_v3.xml.

3.    Log into Ashita4 on the character that will use the XML, allow your inventories time to load completely, and type ‘/ac import Charname_JOB_v3.xml’.

4.    Ashitacast will create a new XML named ‘Charname_JOB.xml’ in ashita/config/ashitacast.

5.  Open ‘Charname_JOB.xml’(I recommend using notepad++) and ctrl-F for <!-- ATTN: to find any potential issues.

6.  Some issues will be easier to correct by editing your original Ashita3 XML, you should correct these in Charname_JOB_v3.xml then perform an import again if necessary.

6.  Other issues should be corrected in the post-conversion XML.  Correct these prior to using the XML.

7.    Repeat as needed for all XMLs.

 

Note that augment tags will only be properly converted if the item that uses that tag is currently on the character executing ‘/ac import’.

The item must be in a bag(safe/storage/etc are fine), and the bag must be loaded(if you just zoned or logged in, it may not be).

The item cannot be in delivery box, outbox, etc.
